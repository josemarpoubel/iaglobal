# RESTRIÇÕES DE ANÁLISE
1. NÃO assuma comportamento — analise o código real
2. Identifique causa raiz, não apenas sintomas
3. Considere edge cases (None, empty lists, exceptions)
4. Verifique:
   - Type errors
   - Null pointer (None) access
   - Index out of bounds
   - Unhandled exceptions
   - Resource leaks
   - Race conditions (se async)

# EXEMPLOS

## Exemplo 1: Null Pointer
Input:
```python
def get_user_name(user):
    return user.name.upper()
```

Erro: `AttributeError: 'NoneType' object has no attribute 'name'`

Output:
```json
{
  "bug_type": "runtime",
  "severity": "high",
  "root_cause": "Função não verifica se user é None antes de acessar atributo",
  "location": {
    "file": "users.py",
    "line": 5,
    "function": "get_user_name"
  },
  "fix": {
    "description": "Adicionar validação de None",
    "code_before": "return user.name.upper()",
    "code_after": "if user is None:\n        return None\n    return user.name.upper()",
    "explanation": "Verifica None antes de acessar atributo"
  },
  "tests": [
    {
      "name": "test_user_none",
      "input": "get_user_name(None)",
      "expected": "None"
    }
  ],
  "prevention": "Sempre valide inputs antes de acessar atributos"
}
```

## Exemplo 2: Index Out of Bounds
Input:
```python
def get_first(items):
    return items[0]
```

Erro: `IndexError: list index out of range`

Output:
```json
{
  "bug_type": "runtime",
  "severity": "medium",
  "root_cause": "Lista vazia não é tratada",
  "location": {
    "file": "utils.py",
    "line": 2,
    "function": "get_first"
  },
  "fix": {
    "description": "Verificar se lista não está vazia",
    "code_before": "return items[0]",
    "code_after": "return items[0] if items else None",
    "explanation": "Retorna None para lista vazia"
  },
  "tests": [
    {
      "name": "test_empty_list",
      "input": "get_first([])",
      "expected": "None"
    }
  ],
  "prevention": "Sempre valide collections antes de indexar"
}
```
