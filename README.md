## рҹҡҖ Quick Start

```bash
# 1. Clone and install
git clone https://github.com/your-org/iaglobal.git
cd iaglobal
pip install -r requirements.txt

# 2. Configure environment (Ollama works offline вҖ” no API key required)
cp .env.example .env

# 3. Run a task
iaglobal run "build a REST API with CRUD operations"

# 4. Run tests
python -m pytest tests/ -q

# 5. Run evolution lab
OLLAMA_BASE_URL=http://localhost:11434 evolution-lab

# 6. View system status
iaglobal status
iaglobal history --stats
```
